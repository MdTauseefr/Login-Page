# Login Page

A modern and responsive Login Page built using HTML, CSS, and JavaScript. This project provides a clean user interface for user authentication and demonstrates front-end development skills such as responsive design, form validation, and UI styling.

## 🚀 Features

* Responsive design for desktop and mobile devices
* Clean and modern user interface
* Email and password input fields
* Client-side form validation
* Social login buttons (Google & Facebook)
* Smooth hover and transition effects
* Easy to customize and extend

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

├── index.html
├── style.css
